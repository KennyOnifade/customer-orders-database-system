# customer-orders-database-system

# 📊 Customer Orders & Salesperson Database System (SQL Server + Python Analysis)

## 📌 Overview
This project demonstrates a complete database system built using SQL Server and Python.

It includes:
- Database design in 3NF
- SQL implementation (DDL & DML)
- Python integration using pyodbc
- Data analysis using pandas
- Data visualization using matplotlib

---

## 🛠️ Technologies Used
- SQL Server (SSMS)
- Python (pandas, matplotlib, pyodbc)
- Jupyter Notebook

---

## 🗄️ Database Structure
The system consists of 5 tables:
- SalesPersons
- Customers
- Orders
- Items
- OrderDetails

---

## 📈 Features
- Insert records using SQL and Python
- Views for business insights
- Function for currency conversion (USD → Euro)
- Stored procedure for inserting salespersons
- Sales analysis by customer and state
